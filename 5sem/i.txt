Есть еще два сискола:
1)int msgsnd(int id (идент очереди, как в райте), void* msg, int size, int flag (например IPC_NOOWAIT))
2)int msgrcv(int id, void* msg, int size, long type, int flag)
Что такое long type?
Структура сообщенгия
struct msg
{
	long type; have to be >0 если <= - отправка вернет ошибку
ВАЖНО: Размер это не размер струкуры, это payload (?) (если передаем толькот тип. то сайз равен нулю)
}
type = 0 первое
type > 0 первое конкретного типа type
type < 0 вроде как >= abs(type)
